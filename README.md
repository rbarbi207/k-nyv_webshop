# Könyv nyilvántartás
A ***Könyv nyilvántartás*** program lehetővé teszi a felhasználó számára könyveinek feljegyzését, és azoknak szabadon való kezelését.
## Használat
**Keresés funkció** - használatával rákereshet bármelyik feljegyzett könyv címére

**Fordító funkció** - magyar és angol nyelv közötti váltás

**Rendezés funkció** - módosítás dátuma szerinti rendezés (csökkenő / növekvő)

**Létrehozás (+)** - a kötelező mezők kitöltésével feljegyezhető bármilyen könyv

**Törlés** - adott könyv törlése

**Módosítás** - adott könyv adatainak módosítása
##
A programot készítette: ***Rozs Barbara***, ***Mezei Renáta***
